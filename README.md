# genlayer-task-reward-tutorial
Beginner tutorial for building a Task Reward dApp on GenLayer

## 🚀 What You’ll Learn

- GenLayer basics  
- Optimistic Democracy Consensus  
- Equivalence Principle  
- Python Intelligent Contracts  
- Frontend interaction using genlayer-js  


## 🧠 Key Concepts

### Optimistic Democracy Consensus
Tasks are marked as completed optimistically and can be challenged by others.

### Equivalence Principle
Multiple validators simulate the same result. If results match, it is accepted.

## ⚙️ How It Works

3. The system verifies completion  
4. Reward is given
